## Quando um doc vai para cá

- O doc descrevia um estado ou plano de um momento específico que hoje é passado.
- O doc foi substituído por uma versão viva em outro local (apontado no banner "ARQUIVADO" do próprio arquivo).
- O doc mantém valor como "por que fizemos assim" — mas não serve para guiar decisões atuais.

## Nunca apagar arquivos daqui

Esta pasta é write-once. Se um doc daqui for realmente obsoleto ao ponto de confundir, adicionar nota no cabeçalho explicando — nunca deletar do histórico.
